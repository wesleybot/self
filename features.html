主要功能

【V2功能介紹】

  • 🔐 授權機制：獲取「專屬金鑰」解鎖下方功能
  • 自動從detail跳轉至game
  • 自動依日期選擇場次
  • 客製化只顯示想要的區域 / 價位
  • 自動篩選可用區域（隱藏不良／指定區域）
  • 自動填入「票張數」與「同意條款」
  • 自動輸入驗證問題答案（例：會員優先購會員碼 OR 銀行優先購）

【V3（測試階段中）嶄新 UI 設計，操作更直覺、便利】

  • 可勾選是否啟動輔助功能 / 設定自動啟動輔助功能時間點
  • 優化選日期，可選擇「日期」OR「場次名稱」
  • 選擇日期可自動選上到下 / 下到上
  • 新增自動電腦選位 / 設定票價或區域選位 / 精準選位
  • 新增自動選擇餘票剩餘最多 / 由上到下選 / 由下到上選
  • 優化「選擇張數」以及「同意條款」
  • 新增OCR自動辨識驗證碼（準確率提升中）
  • 新增自動送出功能

安裝說明

  1. 在首頁下方有 Chrome 擴充下載連結，直接前往 Chrome 商店頁面。
  2. 點擊 新增至 Chrome，並在擴充功能頁面啟用。
  3. 安裝完成後，點擊工具圖示進入設定。
